Cluster Analysis on Framingham Heart Study Data

Project Overview

This project involved performing cluster analysis on the Framingham Heart Study data, a dataset that includes health-related information of individuals. The goal was to group individuals into distinct clusters based on their risk factors for heart disease.

Approach & Methodology

  • Data cleaning and preprocessing to handle missing values.
  • Normalization of data to ensure equal weight across features.
  • Application of K-means clustering algorithm to identify patterns.
  • Visualizing the results using PCA for dimensionality reduction.

Visualizations & Insights

The cluster analysis revealed 3 main groups of individuals, each with varying levels of risk for heart disease. These insights can be useful for targeted healthcare interventions.
